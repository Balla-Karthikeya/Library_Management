# Project develeopment steps 
    1. create folder 
    2. create virtual environment
    3. activate virtual environment
    4. add pre-requirements to requirements.txt
    5. install pre-requirements 
    6. add virtual environment name to gitignore file 
    7. implement library system 
    
***Execute following commands in command prompt***
**step 1:** 
1. create folder command: <br>
```mkdir library```

**step 2**
create virtual environment: <br>
```python -m venv Library_env```

**step 3**
activate virtual environment<br>
```Library_env\Scripts\activate```

- create  ```requirements.txt``` file

**step 4**
add pre-requirements to requirements.txt

**step 5**
install pre-requirements 
```pip install -r requirements.txt```

- create ```gitignore``` file

**step 6**
add virtual environment name to gitignore file 

**step 7**
implement library system
